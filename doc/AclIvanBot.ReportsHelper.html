    Functions

    fetch_name(users, user_id)

    Specs

    fetch_name(map, String.t) :: String.t

    Fetches user by its id Returns user.profile.first_name or user.name in case user.profile.first_name is not present

    Params

    • users: list of users
    • user_id: user id

    Examples

    iex> AclIvanBot.ReportsHelper.fetch_name(%{“U32” => %{ profile: %{ first_name: ‘Ivan’ }}}, “U32”) “IVAN”

    iex> AclIvanBot.ReportsHelper.fetch_name(%{“U33” => %{ name: ‘Petr’ }}, “U33”) “PETR”
